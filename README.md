# 🚗 Road Accident Analysis Project

## Overview
End-to-end data analysis and ML project using the US Accidents dataset (7.7 million records).

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (6 charts)
- Machine Learning Model (Random Forest - 79% accuracy)
- Interactive Web App (Streamlit)

## Tech Stack
Python, Pandas, Matplotlib, Seaborn, Scikit-learn, Streamlit

## Dataset
US Accidents (2016-2023) - Kaggle

## How to Run
pip install -r requirements.txt
streamlit run app.py
